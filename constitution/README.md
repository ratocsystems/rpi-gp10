# RPi-GP10T / RPi-GP10M 各部名称と説明  
- RPi-GP10T: 端子台モデル  
- RPi-GP10M: MILコネクタモデル  
  
___  
## 1. 基板構成  
製品基板の各部名称は以下のとおりです。

![board](./img/GP10_board.png)

| No | 名称 | 
|:-----:|:-----|
|1|GPIO 40PINコネクタ|
|2|COMMON電源端子|
|3|出力端子 8ch ※RPi-GP10T|
|4|入力端子 8ch ※RPi-GP10T|
|5|ストローブ出力端子 / トリガー入力端子(各1ch) ※RPi-GP10T|
|6|MILコネクタ (20PIN) ※RPi-GP10M|
___
